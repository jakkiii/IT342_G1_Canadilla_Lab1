# Screenshots for Documentation

This folder should contain screenshots of the running web application:

## Required Screenshots

1. **register_page.png**
   - Screenshot of the user registration page
   - Should show all form fields (username, email, password, etc.)

2. **login_page.png**
   - Screenshot of the login page
   - Should show username/email and password fields

3. **dashboard_page.png**
   - Screenshot of the user dashboard/profile page
   - Should show user information display

4. **logout_demo.png**
   - Screenshot demonstrating logout functionality
   - Can show logout button or redirect after logout

## How to Take Screenshots

1. Run the backend: `cd backend && mvn spring-boot:run`
2. Run the web app: `cd web && npm start`
3. Register a test user
4. Take screenshots of each page
5. Save them in this folder with the names above
6. Update the FRS document to include these screenshots

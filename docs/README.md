# Documentation Folder

This folder contains all documentation for the User Authentication System project.

## Contents

### 1. Functional Requirements Specification (FRS)
- **File**: `FRS_UserAuth_System.md`
- Contains complete functional requirements, system architecture, and UI screenshots

### 2. Entity Relationship Diagram (ERD)
- **File**: `ERD_Diagram.md`
- Shows database schema and relationships

### 3. UML Diagrams
- **Class Diagram**: `UML_Class_Diagram.md`
- **Sequence Diagrams**: `UML_Sequence_Diagrams.md`
  - User Registration Flow
  - User Login Flow
  - Get User Profile Flow

### 4. Screenshots
Screenshots should be placed in the `/docs/screenshots` folder:
- `register_page.png` - Register page UI
- `login_page.png` - Login page UI
- `dashboard_page.png` - Dashboard/Profile page UI
- `logout_confirmation.png` - Logout functionality demo

## How to Update Documentation

1. Take screenshots of your running web application
2. Save them in the `/docs/screenshots` folder
3. Update the FRS document with the screenshots
4. Export FRS to PDF if required

## Notes

- Screenshots must be taken from the actual running application
- All diagrams use Mermaid syntax for easy editing
- Keep documentation updated with each major change
